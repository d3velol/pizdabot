# PizdaUserBot
## Installation:
`git clone https://github.com/d3velol/pizdabot`
`cd pizdabot`
`python venv env`
`. env/bin/activate`
Linux: `pip install -r requirements.txt`
Windows: `.\env\Scripts\activate`

## Run:
`python main.py`

## Commands: 
`.help`
